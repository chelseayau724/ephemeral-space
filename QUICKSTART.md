# 快速开始指南

## 环境要求

- Node.js 18+
- npm 或 yarn

## 安装和运行

### 1. 安装依赖

\`\`\`bash
npm install
\`\`\`

### 2. 初始化数据库

\`\`\`bash
npx prisma migrate dev
\`\`\`

这会创建 SQLite 数据库文件 `prisma/dev.db`

### 3. 启动开发服务器

\`\`\`bash
npm run dev
\`\`\`

访问 http://localhost:3000

## 快速测试流程

### 场景1: 创建和加入一个空间

1. **打开首页** http://localhost:3000
2. **点击 "创建新空间"**
3. **填写基本信息**
   - 空间名称: "周五小聚"
   - 简介: "随便聊聊"
   - 开始时间: 选择今天稍后的时间
   - 结束时间: 2小时后
4. **配置名片栏位**
   - 必填: 称呼、状态（默认）
   - 可选: 兴趣爱好、今日心情
5. **设置入场问题**（可选）
   - 问题: "今天让你开心的一件事是什么？"
   - 设为选答
6. **选择空间生命周期**
   - 选择 "到时自动消失"
7. **创建完成**
   - 复制四位暗号（如: 3847）

### 场景2: 参与者加入空间

**方法1: 在同一个浏览器**
1. 打开无痕窗口（Command+Shift+N）
2. 访问 http://localhost:3000
3. 输入暗号: 3847
4. 填写名片:
   - 称呼: "小明"
   - 状态: 想聊 🟢
   - 兴趣爱好: "阅读, 咖啡"
   - 入场回答: "喝到了好喝的咖啡"
5. 点击 "进入空间"

**方法2: 在另一个设备**
1. 在手机上访问 http://你的IP:3000
2. 使用相同的暗号进入

### 场景3: 体验空间内部

1. **查看场域状态**
   - 顶部显示剩余时间
   - 在线人数
   - 三种状态的人数分布

2. **浏览他人名片**
   - 卡片式布局
   - 显示OOTD、称呼、状态、标签、入场回答
   - 悬停效果

3. **修改我的名片**
   - 点击底部 "修改名片"
   - 可以更改称呼、状态、更换OOTD
   - 场域状态条会实时更新

4. **离开空间**
   - 点击 "离开空间"
   - 确认后自动创建印记

### 场景4: 查看个人印记

1. **返回首页**
2. **点击 "查看你看过的印记"**
3. **查看历史记录**
   - 日期
   - OOTD小图
   - 称呼
   - 空间暗号

## 数据重置

如果需要清空所有数据重新测试:

\`\`\`bash
# 删除数据库
rm prisma/dev.db

# 重新迁移
npx prisma migrate dev
\`\`\`

## 常见问题

### Q: 暗号验证失败
A: 确保只输入数字，且是4位

### Q: 无法创建空间
A: 检查数据库连接，确保已运行 `npx prisma migrate dev`

### Q: OOTD图片上传失败
A: 确保图片小于 2MB

### Q: 空间已结束但还能进入
A: 系统会显示 "活动时间已过" 提示，但仍然可以查看（这是预期行为）
