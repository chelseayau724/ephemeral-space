## 调试步骤

### 第 1 步：刷新页面
在浏览器中按 **Ctrl + Shift + R** (Windows) 或 **Cmd + Shift + R** (Mac)
强制刷新，确保加载最新代码。

### 第 2 步：打开开发者工具
按 **F12** 打开浏览器开发者工具
切换到 **Console** (控制台) 标签

### 第 3 步：重新尝试创建空间
1. 访问 http://127.0.0.1:3000/create-space
2. 填写空间名称
3. 选择开始时间和结束时间
4. 点击"下一步"
5. Step 2：保持默认，点击"下一步"
6. Step 3：选择一个生命周期选项
7. 点击"创建空间"

### 第 4 步：查看控制台日志

**应该看到：**

#### 前端控制台日志（蓝色信息）：
```

```

```

```

#### 后端控制台日志（运行 `npm run dev` 的终端）：

## 常见错误排查

### 如果看到 "Database error" 或 "Can't reach database server"
**原因：** Prisma 无法连接数据库

**解决：**
```bash
# 检查 .env 文件
cat .env

# 确保有 DATABASE_URL
# 当前是 SQLite，格式应该是：
# DATABASE_URL="file:./dev.db"

# 如果格式不对，修复：
echo 'DATABASE_URL="file:./dev.db"' > .env
```

### 如果看到 "缺少必填字段"
**原因：** 某个必填字段为空

**检查：**
- name 是否填写
- startTime 是否选择
- endTime 是否选择

### 如果看到 "create failed"
**原因：** Prisma 操作失败

**可能原因：**
- 数据库文件损坏
- 权限问题

**解决：**
```bash
# 重新初始化数据库
npx prisma migrate reset --force
```
